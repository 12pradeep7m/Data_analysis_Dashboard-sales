# 🛒 E-Commerce Sales Dashboard (Excel Project)

## 📌 Project Overview

This project analyzes an E-Commerce dataset to uncover key business insights using Microsoft Excel.
An interactive dashboard was built to track sales performance, customer behavior, and revenue trends.

---

## 🎯 Business Problems Solved

This dashboard answers the following key business questions:

* What is the **total revenue generated**?
* Which **month had the highest sales revenue**?
* What is the **Average Order Value (AOV)**?
* Which **product category sells the most units**?
* Which **payment method is most popular**?
* Which months show **peak sales trends**?
